Tweet GeoFinder
================================

An app for Cloudsherpas coding exam. Geolocate tweets within 1 kilometer of desired location in the map.

### Tools
- PHP
- Slimframework
- AngularJS
- Twitter Bootstrap (css)
- LeafletJS
